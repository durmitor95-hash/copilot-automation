# 🤖 Copilot Automation - GitHub Actions Setup

Automatizovani workflow za GitHub Projects sa Copilot integracijom!

## ✨ Šta radi ovaj workflow?

### 1. **Auto-Assign** 🎯
- Kada dodaš `bug` labelu na issue → automatski se dodeljuje tebi
- Štedi vreme, sve je transparentno

### 2. **Auto-Label PR** 🏷️
- Svaki novi Pull Request automatski dobija `review-needed` labelu
- Olakšava praćenje PRs koji čekaju review

### 3. **Welcome Message** 👋
- Svaki novi issue dobija pozdravnu poruku
- Pokazuje da je tvoj tim aktivan i otvoren

## 🚀 Kako početi?

1. **Idi na Issues tab** → Kreiraj novi issue
2. **Dodaj `bug` labelu** → Workflow će te automatski dodeliti
3. **Kreiraj PR** → Automatski će dobiti `review-needed` labelu
